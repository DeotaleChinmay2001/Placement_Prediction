STUDENT PLACEMENT PREDICTION

In this project,We trained a SVM model to predict the whether student can be placed or not in placement . Based on the predction student can prepare himself/herself accordingly .

Instructions for Input Features

  • GENDER TYPE 1 FOR MALE AND O FOR FEMALE
  • SSC PERCENTAGE (0 TO 100)
  • SSC BOARD (1 FOR STATE BOARD 0 FOR OTHER)
  • HSC PERCENTAGE (0 TO 100)
  • HSC BOARD (1 FOR STATE BOARD 0 FOR OTHER)
  • DEGREE PERCENTAGE (0 TO 100)
  • INTERNSHIP (1 FOR YES ,0 FOR NO )
  • EXAM SCORE(0 TO 100)
  • SPECIALISATION YES :1 NO :0
  • MBA PERCENTAGE (0 TO 100)
  • ARTS FIELD 0 FOR NO 1 FOR YES , CHOOSE ONLY ONE AMONG 3
  • COMMERCE FIELD YES :1 NO :0 , CHOOSE ONLY ONE AMONG 3
  • SCIENCE FIELD, CHOOSE ONLY ONE AMONG 3
  • COMMERCE & MANAGEMENT YES :1 NO :0 , EXTERNAL SPECIALIZATION
  • OTHER DEGREE YES :1 NO :0 , ANY COURCES
  • SCIENCE & TECHNOLOGY YES :1 NO :0
